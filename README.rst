Overview
+++++++++++++++++++++

- Manages the HSR configuration files and launch files.

How to use
++++++++++

After releasing HSR's stop buttons,

.. code-block:: bash

                $ sudo systemctl stop docker.hsrb.roscore.service
                $ ros2 launch hsrb_bringup hsrb.launch.py
